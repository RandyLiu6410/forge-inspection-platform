# 提交 code
1. 先用 `git checkout -b [new-branch-name]` 建立新的 branch
2. 開始 coding
3. 完成部分功能就 `git commit` 保留紀錄
4. 重複 2~3 直到功能完成
5. 過程中隨時可以 `git push` 將 code 提交到 github 保存
6. 建立 pull requests 並指定其他成員作 reviewers
7. 等待他人 review 後，就會 merge code 進 develop
